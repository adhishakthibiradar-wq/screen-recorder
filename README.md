# Chrome Screen Recorder

This project was built as **Task 1 of my Software Development Training Program**.
A Chrome-based screen recorder developed using **HTML, CSS, and JavaScript** that allows users to record their screen and microphone, preview the recording, and download it as a WebM file.

## 🎯 Training Task
Build a Chrome-based Screen Recorder that can:

- 🎥 Record the screen
- 🎤 Record microphone audio
- ▶️ Preview the recording
- 💾 Download the recording as a `.webm` file

**Bonus Features Implemented:**
- ⏸️ Pause Recording
- ▶️ Resume Recording
- ⏹️ Stop Recording
- 🕒 Recording Timer
- 🔴 Live Recording Status
- ⚡ Auto Stop when screen sharing ends

## 🚀 How to Run
1. Open the project in VS Code.
2. Use the Live Server extension.
3. Open the application in Google Chrome.
4. Click **Start Recording** and choose a screen or tab to share.

## 📚 Learning Outcomes
- Use the MediaRecorder API
- Capture screen using `getDisplayMedia()`
- Capture microphone using `getUserMedia()`
- Combine multiple media streams
- Handle recording controls (Start, Pause, Resume, Stop)
- Generate downloadable WebM recordings
- Build a responsive user interface using HTML, CSS, and JavaScript
